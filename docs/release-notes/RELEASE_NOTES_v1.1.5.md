## 🔧 Installation

### Fresh Installation
```bash
# Install globally
npm install -g i18ntk@1.1.5

# Verify installation
i18ntk --version  # Should show 1.1.5
```

### Upgrade from Previous Versions
```bash
# Remove old version
npm uninstall -g i18ntk

# Install latest version
npm install -g i18ntk@1.1.5

# Verify upgrade
i18ntk --version  # Should show 1.1.5
```
